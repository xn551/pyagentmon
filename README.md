# PyAgentMon
Become the Master Trainer of PyAgentMon! 
 
A local Python code agent training framework. Train your AI to write, run, debug, and evolve into your exclusive coding partner — fully self‑hosted, sandbox‑safe, and endlessly customizable.Then you can do any cool things, such as 
- writing novels,
- drawing,
- generating models for 3D printing,
- reading literature to design experiments,
- making PPTs or written DOC reports, etc
 
## Core Philosophy
 
- Trainer‑led: You control the architecture, sandbox, context, and rules. Your AI is your partner.
- PCDA loop: Plan → Code → Debug → Adjust → Iterate — real understanding, not just guessing.
- Local‑first: Ollama native models + secure sandbox — private, offline, no external dependencies.
- Modular evolution: Auto‑split functions, multi‑file projects, self‑repair — grows smarter with use.
 
## Features
 
 Secure sandbox execution (isolated directory + dangerous call filtering)
 Automatic PCDA self‑repair loop (fix errors on retry)
 Multi‑file project management (auto‑split into clean functions)
 Long context support (16K+ memory window)
 Pure Python, zero setup friction
 Works with all Ollama local models
